# FundCircle

Digital Pigmy Collection Management Platform for operators, agents, and customers.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the required environment variables (Clerk, Firebase) in Replit Secrets
3. Run the app:
   `npm run dev`
